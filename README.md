# 🐍 MSS Limpa Disco

Aplicativo para limpeza de disco com interface gráfica.

## ✨ Funcionalidades

- 🗑️ Limpeza da lixeira
- 📁 Arquivos temporários
- 🌐 Cache de navegadores (Chrome, Edge, Firefox)
- 🖼️ Cache de miniaturas
- 📊 Interface moderna com tema escuro
- 🚀 Limpeza em thread separada (não trava)

## 📋 Requisitos

- Python 3.8+
- Windows (usa recursos específicos do sistema)

## 🔧 Instalação

```bash
# Clone o repositório
git clone https://github.com/mssacramento/mss-limpa-disco.git

# Entre na pasta
cd mss-limpa-disco

# Crie ambiente virtual
python -m venv venv

# Ative
venv\Scripts\activate

# Instale dependências
pip install -r requirements.txt

# Execute
python main.py
